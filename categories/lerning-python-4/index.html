第十五篇：函数

#####1.def

  • 函数的关键字是def。def可以出现在if, while 这些语句内。当Python检查到def时，它创建一个函数对象。并使def中的函数名指向这个对象。
  • lambda 创建一个无名函数对象。
  • return 表示函数返回。
  • yield 抛出一个返回值。但是不退出函数。
  • global 默认变量是local的。使用global，可以是变量属于全局的。
  • nonlocal 3.0引入的类似于global的标识符。
