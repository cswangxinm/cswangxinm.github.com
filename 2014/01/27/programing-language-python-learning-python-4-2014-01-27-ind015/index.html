#####4. 参数传递
参数传递的规则：

  • 自动赋值到local变量名
  • 函数中改变参数名不影响caller
  • 改变参数内容可能影响到caller

参数传递的原则是：

  • Immutable(如：integers, strings.你不能修改它们) 参数按值传递
  • Mutable 参数按引用传递

#####18. Generator Expressions
