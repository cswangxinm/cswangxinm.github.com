#####1.3/4 == 0？
在python中，每个数都会关联到一个类型上的，3， 4默认为整型。
（python的一个好处是这些类型都不用声明，直接拿来就用。）
这里3/4 == 0 可以类比c中的int类型除法。结果向下取舍，就变成了0.

3.0 / 4.0就可以了。python看到小数点，自然明白这是float类型（flaot类型在python中类似于c中float/double)
