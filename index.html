#####1. Import, from
当import时，python会自动执行import的程序。模块导入的过程：

  1. Find the module’s file.
  2. Compile it to byte code.
  3. Run the module’s code to build the objects it defines.

注意上面的过程只在第一次导入时发生。以后导入后，python仅仅在内存中寻找对应的模块。python将所有模块保存在一个表中：sys.modules。导入模块时，python会检查该模块是不是在其中。如果不在其中，会发生上面的3个步骤。在编译模块之前，python检查文件的时间戳。如果.pyc文件不比源文件的.py文件老，则会忽略第二步。还有一点要注意的是，import会忽略导入模块的路径名。为了区别导入时的运行行为和作为主文件运行的运行行为。可以使用__name__ 和 __main__ 来限定。

集成学习的三中方法

1 Bagging (Boostrap Aggregation)

通过叠加训练出多个分类器来降低判别误差方差。
2 Boosting

通过调整样本权重生成多个分类器进行判别。

3 Stacking

首先使用不同的算法学习原始数据，得到第一层分类器（first-level individual learners). 然后将第一层分类器输入到第二层分类器(meta-learner)再进行分类。
